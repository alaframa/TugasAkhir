Project Overview

This project involves the development of a real-time IoT-based energy monitoring and control system using ESP32 nodes equipped with 3 PZEM sensor modules to detect 3-phase electricity. The system includes a remote server for gathering, monitoring, and logging data from the nodes.

Node Technologies

  • FreeRTOS
  • Flash
  • Multi-core Processing
  • Network Management
  • Remote Configuration

Server Technologies

  • Node-RED
  • Flask
  • Python
  • Mosquitto
  • MySQL
  • InfluxDB
  • Grafana

Features

  • User Authentication with email-based token sending
  • Handling both MQTT and HTTP data from nodes
